# 💰 MGA Finance Tracker

Une application de gestion financière professionnelle avec Streamlit.

## 🚀 Fonctionnalités

- 🔐 Authentification sécurisée (inscription/connexion)
- 📊 Tableau de bord interactif
- 💰 Gestion des transactions (revenus, dépenses, prêts)
- 📝 Gestion des projets
- 📓 Cahier de notes avec suivi des estimations
- 🎨 Interface moderne avec animations
- 📈 Graphiques interactifs avec Plotly
- 🔍 Filtres précis et personnalisés
